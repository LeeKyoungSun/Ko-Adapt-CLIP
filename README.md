# ko-adapt-clip

Korean adaptation analysis for CLIP and multimodal models.

## Goals

- Compare English vs Korean CLIP performance
- Analyze tokenizer fragmentation in Korean
- Evaluate cultural concept understanding
- Few-shot Korean adaptation experiments

## Datasets

- MSCOCO
- KoCOCO
- AIHub Korean Culture Images
- Custom curated Korean dataset

## Models

- CLIP ViT-B/32
- CLIP ViT-L/14
- LLaVA variants
